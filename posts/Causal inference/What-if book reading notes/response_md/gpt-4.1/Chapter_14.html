Chapter 14: G-Estimation of Structural Nested Models

本章主要内容为结构嵌套均值模型（SNMM, Structural Nested Mean Models）的g估计（g-estimation），它是三种g方法之一（g-methods，另两种为IP 加权和标准化），用于估计因果效应，特别适合处理时间变化的处理方案，但本章通过固定处理的例子来讲解g-估计的思想和细节。

1. 因果问题回顾 / Causal Question Revisited

主要内容

  • 假设研究戒烟（AAA，1 表示戒烟，0 表示未戒烟）对体重增加（YYY）的平均因果效应。
  • 假设条件交换性在协变量 LLL（性别、年龄、种族、教育、吸烟强度/年限、活动、体重等）成立。
  • 有兴趣的量：

    Average Causal Effect=E[Ya=1,c=0]−E[Ya=0,c=0]\text{Average Causal Effect} = E[Y^{a=1, c=0}] - E[Y^{a=0, c=0}] Average Causal Effect=E[Ya=1,c=0]−E[Ya=0,c=0]

    可条件化到子群，比如 E[Ya=1,c=0∣age=45]−E[Ya=0,c=0∣age=45]E[Y^{a=1, c=0} \mid \text{age}=45] - E[Y^{a=0, c=0} \mid \text{age}=45]E[Ya=1,c=0∣age=45]−E[Ya=0,c=0∣age=45]。

结论

  • 可用边际结构模型（MSM）、标准化或g估计估算群体或某子集合的平均因果效应。

2. 条件交换性回顾 / Exchangeability Revisited

主要内容

  • 条件交换性（analogous to conditional ignorability）核心表达：

    Ya⊥A∣L,for both a=0,1Y^a \perp A \mid L, \quad \text{for both } a=0,1 Ya⊥A∣L,for both a=0,1

    也即：

    Pr[A=1∣Ya=0,L]=Pr[A=1∣L]Pr[A=1 \mid Y^{a=0}, L] = Pr[A=1 \mid L] Pr[A=1∣Ya=0,L]=Pr[A=1∣L]

  • 可用如下logistic模型表述（AAA为二元）：

    \logitPr[A=1∣Ya=0,L]=α0+α1Ya=0+α2L\logit Pr[A=1 \mid Y^{a=0}, L] = \alpha_0 + \alpha_1 Y^{a=0} + \alpha_2 L \logitPr[A=1∣Ya=0,L]=α0​+α1​Ya=0+α2​L

技术要点

  • 在 Ya=0Y^{a=0}Ya=0 真正独立于 AAA 时，α1=0\alpha_1=0α1​=0。g估计正是通过寻找α1=0\alpha_1=0α1​=0的参数值实现的。

3. 结构嵌套均值模型 / Structural Nested Mean Models (SNMM)

主要内容

  • 兴趣在于各LLL层次下的平均因果效应

    E[Ya=1−Ya=0∣L]=E[Ya=1∣L]−E[Ya=0∣L]E[Y^{a=1} - Y^{a=0} \mid L] = E[Y^{a=1} \mid L] - E[Y^{a=0} \mid L] E[Ya=1−Ya=0∣L]=E[Ya=1∣L]−E[Ya=0∣L]

  • 没有交互作用（effect-modification）时，因果效应在LLL内恒定：

    E[Ya−Ya=0∣L]=β1aE[Y^{a} - Y^{a=0} \mid L] = \beta_1 a E[Ya−Ya=0∣L]=β1​a

  • 有效应修饰时，加乘项：

    E[Ya−Ya=0∣L]=β1a+β2aLE[Y^{a} - Y^{a=0} \mid L] = \beta_1 a + \beta_2 aL E[Ya−Ya=0∣L]=β1​a+β2​aL

和MSM关系

  • 不指定E[Ya=0∣V]E[Y^{a=0} \mid V]E[Ya=0∣V]（VVV为协变量）参数的半参数MSM和SNMM有等价表达。
  • "Fine Point 14.1" 细节：关注β1a+β2aV\beta_1 a + \beta_2 aVβ1​a+β2​aV的估计时（只关心平均效应，不关心未处理下均值），半参数模型更鲁棒。

4. 选择偏倚与g估计的配合 / Censoring and G-Estimation

重点

  • g估计只校正混杂，不可直接校正选择（删失）偏倚。需对删失用IP加权后，再行g估计。
  • 非稳定型IP权重（如WC=1/Pr⁡[C=0∣L,A]W^C = 1/\Pr[C=0|L,A]WC=1/Pr[C=0∣L,A]）用于创建“去删失伪总体”。

5. SNMM其他类型与技术点 / SNMM Variants and Technical Points

Technical Point 14.1

  • 乘法结构嵌套均值模型（Multiplicative SNMMs），适用于Y>0Y>0Y>0。
  • 形式：

    log⁡(E[Ya∣A=a,L]E[Ya=0∣A=a,L])=β1a+β2aL\log \left( \frac{E[Y^a|A=a,L]}{E[Y^{a=0}|A=a,L]} \right) = \beta_1 a + \beta_2 aL log(E[Ya=0∣A=a,L]E[Ya∣A=a,L]​)=β1​a+β2​aL

6. 秩保留 (Rank Preservation) 概念

主要内容

  • 若每个LLL分层内干预效应对所有人都相同，则序列/排名不会因AAA而改变——这就是秩保留。
  • 更现实的情况：秩保留不成立，AAA效应因人而异（不要求个人因果效应恒定）。

结论：SNMM和g估计不要求秩保留成立。

7. g-估计核心方法论 / Core Method: G-Estimation

基本步骤

  1. 假设结构嵌套模型 Ya−Ya=0=ψ1aY^a - Y^{a=0} = \psi_1 aYa−Ya=0=ψ1​a。
  2. 用一致性原则，将未观察到的YaY^aYa用观测变量和AAA表达，得到：

    Ya=0=Y−ψ1AY^{a=0} = Y - \psi_1 A Ya=0=Y−ψ1​A

    构造一系列H(ψ†)=Y−ψ†AH(\psi^\dagger) = Y - \psi^\dagger AH(ψ†)=Y−ψ†A作为候选Ya=0Y^{a=0}Ya=0。
  3. 在每个ψ†\psi^\daggerψ†下拟合

    \logitPr[A=1∣H(ψ†),L]=α0+α1H(ψ†)+α2L\logit Pr[A=1 \mid H(\psi^\dagger), L] = \alpha_0 + \alpha_1 H(\psi^\dagger) + \alpha_2 L \logitPr[A=1∣H(ψ†),L]=α0​+α1​H(ψ†)+α2​L

    找到ψ^1\hat{\psi}_1ψ^​1​使得α^1=0\hat{\alpha}_1 = 0α^1​=0（H(ψ1)H(\psi_1)H(ψ1​)与AAA在LLL下独立）。
  4. 置信区间：对于ψ†\psi^\daggerψ†，检验α^1=0\hat{\alpha}_1=0α^1​=0的P值大于0.05所对应区间。

技术要点

  • 截至点 ψ^1\hat{\psi}_1ψ^​1​ 是平均因果效应的g估计。
  • 配合删失用IP权重。置信区间可用bootstrap。

灵敏度分析（Fine Point 14.2）

  • 若LLL不全，α1≠0\alpha_1 \neq 0α1​=0。g估计可设α1\alpha_1α1​为已知非零常数，做灵敏度分析（系统研究未观测混杂影响）。

8. g-估计多参数与计算 / Multivariate G-Estimation

  • 对模型 E[Ya−Ya=0∣A=a,L]=β1a+β2aVE[Y^a - Y^{a=0} |A=a, L] = \beta_1 a + \beta_2 aVE[Ya−Ya=0∣A=a,L]=β1​a+β2​aV，H(β†)=Y−β1†A−β2†AVH(\beta^\dagger) = Y - \beta^\dagger_1 A - \beta^\dagger_2 AVH(β†)=Y−β1†​A−β2†​AV。

  • Logistic 模型

    \logitPr[A=1∣H(β†),L]=α0+α1H(β†)+α2H(β†)V+α3L\logit Pr[A=1 \mid H(\beta^\dagger), L] = \alpha_0 + \alpha_1 H(\beta^\dagger) + \alpha_2 H(\beta^\dagger) V + \alpha_3 L \logitPr[A=1∣H(β†),L]=α0​+α1​H(β†)+α2​H(β†)V+α3​L

    寻找使α1=α2=0\alpha_1 = \alpha_2 = 0α1​=α2​=0的参数（多维搜索）。

  • 对线性模型，β\betaβ 可用封闭解直接算出，无需遍历（见Technical Point 14.2）。

10. 更高维度建模与Bias问题

  • SNMM 主动建模的是LLL层次下的因果效应，少加产品项会bias（MSM不会）。
  • SNMM 可以灵活设定各交互（如 aLjaL_jaLj​），但必须包含所有效应修改项，否则有模型偏倚。
  • 平均因果效应可由所有层次加权平均得出。

结论与实践指引 / Conclusions and Practical Guidance

  • g估计提供了一种利用假想“无处理”下的 counterfactual 的方法，核心是找参数使在L下新构造变量与A独立。
  • SNMM兼容异质效应，但多参数时要全面建模；
  • g估计具有灵敏性分析能力，对未观测混杂可做系统性偏差评估；
  • 具体实现时可用反复拟合logistic模型（或评分方程闭式解）得到g估计点与置信区间；
  • 需在正确处理选择偏倚（删失）的情况下使用g估计，否则结果无法解释为因果效应。

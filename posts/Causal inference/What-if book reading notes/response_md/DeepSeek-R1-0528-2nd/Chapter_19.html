### 第19章 时变处理

19.1 时变处理的因果效应

技术重点（Technical Points）：

  • 核心理论：时变处理（如随时间变化的医疗干预）的因果效应需显式纳入时间维度，与固定处理不同。固定处理的平均因果效应定义为 E[Ya=1]−E[Ya=0]E[Y^{a=1}] - E[Y^{a=0}]E[Ya=1]−E[Ya=0]，而时变处理需通过对比完整处理策略的潜在结果均值来定义。
  • 关键公式：
    • 时变二值处理 AkA_kAk​（k=0,1,…,Kk=0,1,\dots,Kk=0,1,…,K，K=59K=59K=59），表示每月是否接受治疗。
    • 治疗史记为 Aˉk=(A0,A1,…,Ak)\bar{A}_k = (A_0, A_1, \dots, A_k)Aˉk​=(A0​,A1​,…,Ak​)，完整治疗史 Aˉ=AˉK\bar{A} = \bar{A}_KAˉ=AˉK​。
  • 重要概念：
    • 时变处理的因果效应非唯一，需通过对比特定处理策略（如“始终治疗”Aˉ=1ˉ\bar{A}=\bar{1}Aˉ=1ˉ vs “从不治疗”Aˉ=0ˉ\bar{A}=\bar{0}Aˉ=0ˉ）定义。
  • 应用场景：HIV研究中抗逆转录病毒疗法的动态效果评估（治疗随CD4细胞计数变化）。

细节要点（Fine Points）：

  • 简化假设：研究期间无失访或死亡，所有变量测量完美。
  • 示例细节：
    • HIV研究中，Ak=1A_k=1Ak​=1表示第kkk月接受治疗；研究开始时所有个体A_{-1}=0。
    • 结局YYY在随访结束（K+1=60K+1=60K+1=60月）时测量，值越大表示健康状况越好。
  • 局限性：单时间点对比（如E[Yak=1]−E[Yak=0]E[Y^{a_k=1}] - E[Y^{a_k=0}]E[Yak​=1]−E[Yak​=0]）仅反映特定时点kkk的效应，而非整个时变处理的效应。

19.2 处理策略

技术重点（Technical Points）：

  • 核心理论：处理策略（strategy）是分配治疗的规则，分为静态策略（aˉ\bar{a}aˉ）和动态策略（ggg）。动态策略中，aka_kak​依赖于时变协变量Lˉk\bar{L}_kLˉk​（如CD4细胞计数）。
  • 重要概念：
    • 静态策略：所有个体在相同时间点接受相同治疗（如aˉ=(1,0,1,0… )\bar{a}=(1,0,1,0\dots)aˉ=(1,0,1,0…)）。
    • 动态策略：治疗分配依赖个体协变量演化（如“当Lk=1L_k=1Lk​=1时开始治疗”）。
  • 因果效应定义：通过对比策略的潜在结果均值（如E[Yaˉ]−E[Yaˉ′]E[Y^{\bar{a}}] - E[Y^{\bar{a}'}]E[Yaˉ]−E[Yaˉ′]）定义，策略数量随KKK指数增长（2K2^K2K种可能）。

细节要点（Fine Points）：

  • 策略类型（Fine Point 19.1）：
    • 确定性策略：分配固定治疗值（如gk(aˉk−1,lˉk)g_k(\bar{a}_{k-1}, \bar{l}_k)gk​(aˉk−1​,lˉk​)指定AkA_kAk​为0或1）。
    • 随机策略：分配治疗概率（如“每月以概率0.3治疗”）。
  • 最优策略：最大化E[Yg]E[Y^g]E[Yg]的策略，通常为动态策略（如药物毒性时停药）。
  • 示例：HIV研究中，动态策略ggg根据CD4计数分配治疗，而静态策略aˉ\bar{a}aˉ忽略协变量。

19.3 序贯随机实验

技术重点（Technical Points）：

  • 核心理论：序贯随机实验中，AkA_kAk​的分配概率仅依赖过去治疗史Aˉk−1\bar{A}_{k-1}Aˉk−1​和协变量史Lˉk\bar{L}_kLˉk​（图19.1-19.2）。若分配依赖未测混杂UkU_kUk​（图19.3），则存在未测量混杂。
  • 重要概念：
    • 观察性研究通常对应图19.2（有测量混杂）或图19.3（有未测量混杂），无法通过数据区分。
  • 应用场景：实验设计中，治疗分配基于可观测变量（如CD4计数），而观察性研究中分配概率需从数据估计。

细节要点（Fine Points）：

  • 实验设计：
    • 图19.1：AkA_kAk​仅依赖Aˉk−1\bar{A}_{k-1}Aˉk−1​（如既往未治疗者以概率0.5分配治疗）。
    • 图19.2：AkA_kAk​依赖Aˉk−1\bar{A}_{k-1}Aˉk−1​和Lˉk\bar{L}_kLˉk​（如CD4计数低者治疗概率更高）。
  • 识别限制：图19.3中AkA_kAk​依赖UkU_kUk​，因果效应不可识别。

19.4 序贯可交换性

技术重点（Technical Points）：

  • 核心理论：时变处理的因果推断需序贯可交换性：Yg⊥ ⁣ ⁣ ⁣⊥Ak∣Aˉk−1=g(Aˉk−2,Lˉk−1),LˉkY^g \perp\!\!\!\perp A_k \mid \bar{A}_{k-1} = g(\bar{A}_{k-2}, \bar{L}_{k-1}), \bar{L}_kYg⊥⊥Ak​∣Aˉk−1​=g(Aˉk−2​,Lˉk−1​),Lˉk​（对所有kkk和策略ggg）。此条件在序贯随机实验（图19.2）中成立。
  • 关键公式：
    • 两时间点示例：Yg⊥ ⁣ ⁣ ⁣⊥A0∣L0Y^g \perp\!\!\!\perp A_0 \mid L_0Yg⊥⊥A0​∣L0​ 且 Yg⊥ ⁣ ⁣ ⁣⊥A1∣A0=g(L0),L0,L1Y^g \perp\!\!\!\perp A_1 \mid A_0=g(L_0), L_0, L_1Yg⊥⊥A1​∣A0​=g(L0​),L0​,L1​。
  • 识别条件：
    • 图19.2中所有策略ggg的E[Yg]E[Y^g]E[Yg]可识别；图19.3中不可识别。
    • 需额外满足正性（positivity）和一致性（consistency）条件（Technical Point 19.2）。

细节要点（Fine Points）：

  • 正性与一致性（Technical Point 19.2）：
    • 正性：若fAˉk−1,Lˉk(aˉk−1,lˉk)≠0f_{\bar{A}_{k-1},\bar{L}_k}(\bar{a}_{k-1}, \bar{l}_k) \neq 0fAˉk−1​,Lˉk​​(aˉk−1​,lˉk​)=0，则fAk∣Aˉk−1,Lˉk(ak∣aˉk−1,lˉk)>0f_{A_k \mid \bar{A}_{k-1},\bar{L}_k}(a_k \mid \bar{a}_{k-1}, \bar{l}_k) > 0fAk​∣Aˉk−1​,Lˉk​​(ak​∣aˉk−1​,lˉk​)>0。
    • 一致性：若个体实际治疗与策略ggg一致，则Yg=YY^g = YYg=Y；静态策略下若A=aˉA = \bar{a}A=aˉ，则Yaˉ=YY^{\bar{a}} = YYaˉ=Y。
  • 部分识别：图19.4中，静态策略E[Yaˉ]E[Y^{\bar{a}}]E[Yaˉ]可识别，但依赖L1L_1L1​的动态策略不可识别。

19.5 部分策略下的可识别性

技术重点（Technical Points）：

  • 核心理论：使用SWIGs（单例干预图）检验可识别性：
    • 静态策略下，静态序贯可交换性 Yaˉ⊥ ⁣ ⁣ ⁣⊥Ak∣Aˉk−1=aˉk−1,LˉkY^{\bar{a}} \perp\!\!\!\perp A_k \mid \bar{A}_{k-1} = \bar{a}_{k-1}, \bar{L}_kYaˉ⊥⊥Ak​∣Aˉk−1​=aˉk−1​,Lˉk​ 可识别E[Yaˉ]E[Y^{\bar{a}}]E[Yaˉ]（图19.5-19.6）。
    • 动态策略需更强条件（图19.5成立，图19.6不成立）。
  • 重要概念：
    • 图19.5：静态和动态策略均可识别。
    • 图19.6：仅静态策略可识别（因路径A0←W0→L1gA_0 \leftarrow W_0 \rightarrow L_1^gA0​←W0​→L1g​导致动态策略不可识别）。

细节要点（Fine Points）：

  • SWIGs应用（Fine Point 19.2）：
    • 干预节点（如a0a_0a0​）为常量，路径中包含常量的路径被阻塞（如A_1}^{a_0} \leftarrow a_0 \rightarrow L_1^{a_0} \leftarrow U_1 \rightarrow Y^{a_0,a_1}）。
  • 反例：图19.11中（含L1→YL_1 \rightarrow YL1​→Y），所有策略均不可识别（图19.12）。

19.6 时变混杂与混杂因子

技术重点（Technical Points）：

  • 核心理论：时变混杂需调整Lˉk\bar{L}_kLˉk​以阻断AkA_kAk​与YYY的后门路径（如Ak←Lk←Uk→YA_k \leftarrow L_k \leftarrow U_k \rightarrow YAk​←Lk​←Uk​→Y）。未测量混杂（UkU_kUk​）存在时，效应不可识别。
  • 重要概念：
    • 时变混杂因子：影响后续治疗且与结局共享原因的协变量（如CD4计数）。
    • 充分测量Lˉk\bar{L}_kLˉk​可控制混杂（图19.5），但无法实证验证是否所有混杂已测量。

细节要点（Fine Points）：

  • 混杂定义（Fine Point 19.3）：
    • 若E[Yaˉ∣L0]≠E[Y∣A=aˉ,L0]E[Y^{\bar{a}} \mid L_0] \neq E[Y \mid A=\bar{a}, L_0]E[Yaˉ∣L0​]=E[Y∣A=aˉ,L0​]，存在时变混杂。
    • 无时变混杂的充分条件：Yaˉ⊥ ⁣ ⁣ ⁣⊥Ak∣Aˉk−1=aˉk−1Y^{\bar{a}} \perp\!\!\!\perp A_k \mid \bar{A}_{k-1} = \bar{a}_{k-1}Yaˉ⊥⊥Ak​∣Aˉk−1​=aˉk−1​（图19.1）。
  • 方法限制：即使所有混杂已测量，传统调整方法仍可能有偏，需用g-方法（第21章）。

章节逻辑关联

本章扩展了前两部分固定处理的因果推断框架，引入时变处理的复杂性。核心在于：

  1. 通过处理策略形式化动态干预；
  2. 依赖序贯可交换性和时变混杂控制实现因果效应识别；
  3. 强调SWIGs等工具在可识别性检验中的关键作用。
    后续章节将介绍g-公式等具体方法解决时变混杂调整问题。

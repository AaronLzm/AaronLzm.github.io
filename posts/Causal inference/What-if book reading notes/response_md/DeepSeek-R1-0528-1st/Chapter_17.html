### 第17章：因果生存分析（Causal Survival Analysis）总结

17.1 风险与危险（Hazards and Risks）

  • 核心问题：估计处理（如戒烟）对事件发生时间（如死亡时间）的因果效应，考虑行政删失（administrative censoring）。
  • 关键概念：
    • 行政删失：研究结束时部分个体未发生事件，其生存时间未知（仅知 > 研究结束时间）。
    • 生存概率（Survival Probability）：Pr[T>k]Pr[T > k]Pr[T>k]，个体在时间 kkk 前未发生事件的比例。
    • 风险（Risk/Cumulative Incidence）：Pr[T≤k]=1−Pr[T>k]Pr[T \leq k] = 1 - Pr[T > k]Pr[T≤k]=1−Pr[T>k]，时间 kkk 前发生事件的累积概率。
    • 危险（Hazard）：Pr[T=k∣T>k−1]Pr[T = k | T > k-1]Pr[T=k∣T>k−1]，在时间 k−1k-1k−1 存活者中，kkk 时发生事件的概率（离散时间）。
  • 危险比（Hazard Ratio）的局限性：
    • 随时间变化，常被加权平均报告（如Cox模型），解释困难。
    • 因"conditioning on a post-treatment variable"（如存活至时间 kkk）可能引入选择偏倚（见图17.3）。

17.2 从危险到风险（From Hazards to Risks）

  • 数据格式：
    • 宽格式（Wide）：每行代表一个个体。
    • 人时格式（Person-time）：每行代表一个个体在一个时间点的状态（如每月一行）。
  • 事件指标：Dk=I(T≤k)D_k = I(T \leq k)Dk​=I(T≤k)，T≤kT \leq kT≤k 时取1，否则取0。
  • 生存概率计算：

    Pr[Dk=0]=∏m=1kPr[Dm=0∣Dm−1=0]=∏m=1k(1−λm)Pr[D_k = 0] = \prod_{m=1}^{k} Pr[D_m = 0 | D_{m-1} = 0] = \prod_{m=1}^{k} (1 - \lambda_m) Pr[Dk​=0]=m=1∏k​Pr[Dm​=0∣Dm−1​=0]=m=1∏k​(1−λm​)

    其中 λm=Pr[Dm=1∣Dm−1=0]\lambda_m = Pr[D_m = 1 | D_{m-1} = 0]λm​=Pr[Dm​=1∣Dm−1​=0] 为 mmm 时刻危险。
  • 估计方法：
    • 非参数：Kaplan-Meier 估计量（乘积极限估计）。
    • 参数模型：逻辑回归拟合危险，例如：

      logit Pr[Dk+1=1∣Dk=0,A]=θ0,k+θ1A+θ2A×k+θ3A×k2\text{logit } Pr[D_{k+1} = 1 | D_k = 0, A] = \theta_{0,k} + \theta_1 A + \theta_2 A \times k + \theta_3 A \times k^2 logit Pr[Dk+1​=1∣Dk​=0,A]=θ0,k​+θ1​A+θ2​A×k+θ3​A×k2

      θ0,k\theta_{0,k}θ0,k​ 为时间相关截距（如 θ0+θ4k+θ5k2\theta_0 + \theta_4 k + \theta_5 k^2θ0​+θ4​k+θ5​k2）。

17.3 删失的重要性（Why Censoring Matters）

  • 随机进入研究（Staggered Entry）：个体进入时间不同导致行政删失时间 kendk_{\text{end}}kend​ 不同。
  • 删失指标：Ck=I(行政结束时间≤k)C_k = I(\text{行政结束时间} \leq k)Ck​=I(行政结束时间≤k)。
  • 正确估计生存概率：在无随机删失假设下：

    Pr[Dk=0∣A=a]=∏m=1kPr[Dm=0∣Dm−1=0,Cm=0,A=a]Pr[D_k = 0 | A = a] = \prod_{m=1}^{k} Pr[D_m = 0 | D_{m-1} = 0, C_m = 0, A = a] Pr[Dk​=0∣A=a]=m=1∏k​Pr[Dm​=0∣Dm−1​=0,Cm​=0,A=a]

  • 忽略删失的偏倚：仅用未删失个体计算生存概率会导致选择偏倚（当处理影响生存时间时）。

17.4 边际结构模型的IP加权（IP Weighting of MSMs）

  • 目标：估计反事实生存曲线 Pr[Dk+1a=0]Pr[D^{a}_{k+1} = 0]Pr[Dk+1a​=0]（a=1a=1a=1 和 a=0a=0a=0）。
  • 假设：可交换性（exchangeability）、一致性（consistency）、正性（positivity）。
  • 步骤：
    1. 估计IP权重：
      • 拟合处理模型 Pr[A=1∣L]Pr[A=1 | L]Pr[A=1∣L]，计算稳定权重 SWA=Pr[A=1]f(A∣L)SW^A = \frac{Pr[A=1]}{f(A|L)}SWA=f(A∣L)Pr[A=1]​。
    2. 加权拟合危险模型：用人时数据拟合加权逻辑模型：

      logit Pr[Dk+1=1∣Dk=0,A]=β0,k+β1a+β2a×k+β3a×k2\text{logit } Pr[D_{k+1} = 1 | D_k = 0, A] = \beta_{0,k} + \beta_1 a + \beta_2 a \times k + \beta_3 a \times k^2 logit Pr[Dk+1​=1∣Dk​=0,A]=β0,k​+β1​a+β2​a×k+β3​a×k2

    3. 计算生存曲线：将估计危险代入生存概率乘积公式。
  • 应用示例：戒烟对死亡率影响（NHEFS数据），调整后120个月生存率差异不显著（80.7% vs 80.5%）。

17.5 参数化g公式（Parametric g-Formula）

  • 公式：

    Pr[Dk+1a=0]=∑lPr[Dk+1=0∣L=l,A=a]Pr[L=l]Pr[D^{a}_{k+1} = 0] = \sum_l Pr[D_{k+1} = 0 | L = l, A = a] Pr[L = l] Pr[Dk+1a​=0]=l∑​Pr[Dk+1​=0∣L=l,A=a]Pr[L=l]

  • 步骤：
    1. 估计条件危险：拟合含协变量 LLL 的危险模型（如逻辑回归）。
    2. 预测条件生存概率：计算 Pr[Dk+1=0∣L=l,A=a]Pr[D_{k+1} = 0 | L = l, A = a]Pr[Dk+1​=0∣L=l,A=a]。
    3. 标准化：对 lll 求平均（通过扩展数据集、预测和平均）。
  • 应用示例：结果与IP加权类似（120个月生存率80.4% vs 80.6%）。

17.6 结构嵌套模型的G估计（G-Estimation of SNMs）

  • 加速失效时间模型（AFT）：
    • 定义：Ti0=Tiexp⁡(ψ1Ai+ψ2AiLi)T^{0}_i = T_i \exp(\psi_1 A_i + \psi_2 A_i L_i)Ti0​=Ti​exp(ψ1​Ai​+ψ2​Ai​Li​)，ψ1<0\psi_1 < 0ψ1​<0 表示处理延长生存时间。
    • 秩保持问题：模型为确定性和秩保持（不现实），但G估计可推广至非秩保持模型。
  • G估计步骤：
    1. 构造反事实生存时间：Hi(ψ†)=Tiexp⁡(ψ†Ai)H_i(\psi^\dagger) = T_i \exp(\psi^\dagger A_i)Hi​(ψ†)=Ti​exp(ψ†Ai​)。
    2. 人工删失（Artificial Censoring）：排除反事实处理下会被删失的个体（定义 Δ(ψ)\Delta(\psi)Δ(ψ) 指标）。
    3. 估计方程：寻找 ψ\psiψ 使 Δ(ψ)\Delta(\psi)Δ(ψ) 与 AAA 在给定 LLL 下独立。
  • 应用示例：生存时间比率点估计为1.05（95% CI: 0.77-1.25），无显著效应。

结论

  1. 生存分析核心：处理行政删失，使用生存概率、风险、危险等指标，避免依赖危险比。
  2. 因果推断方法：
    • IP加权：通过权重调整混杂，需正确指定处理模型和危险模型。
    • g公式：直接标准化，需正确指定条件危险模型。
    • G估计（AFT）：建模生存时间比率，计算复杂但更贴近机制解释。
  3. 关键挑战：
    • 竞争风险（Competing Risks）：需谨慎处理（如Fine Point 17.1）。
    • 时变处理：需扩展至Part III的时变模型。
  4. 实际应用：戒烟对10年生存率无显著影响（调整后差异<2%），各方法结果一致。
